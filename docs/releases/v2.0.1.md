## 릴리스 목적

V2.0.0의 Foundry-first bounded agent 구조와 공개 API는 유지하면서, 최종 소스 감사에서 발견한 숫자/조사 correctness 문제와 Ollama 생성 상한을 보완합니다. 공개 PyPI 사용자가 설치·실행·결과를 더 빨리 확인할 수 있도록 README와 source distribution도 함께 정리합니다.

## 수정 범위

### Correctness

- `customer만`, `"customer"만`, `C++만`, `account이`, `계정이`처럼 canonical term이나 일반 명사에 붙은 한국어 조사를 숫자로 오인하지 않습니다.
- `커스터머만 → customer만` 결과에 mapper를 다시 적용해도 같은 문자열을 유지합니다.
- 호출별 `threshold`는 생성자와 동일하게 finite, non-bool, `0..1` 값만 허용합니다.

### Provider guardrail

- Ollama structured-output 요청에 기본 `max_output_tokens=2048`을 추가하고 `num_predict`로 전달합니다.
- Qwen 계열을 포함한 reasoning model이 JSON 대신 thinking budget을 소비하지 않도록 `think=False`를 전달합니다.
- Foundry/Ollama extra 미설치 오류는 PyPI package 설치 명령을 안내합니다.

### Packaging과 문서

- sdist에 eval runner, golden cases, vocabulary, tests, docs와 운영 workflow를 포함합니다.
- 게시 workflow가 annotated tag, `pyproject.toml` version과 runtime `__version__`을 함께 검증합니다.
- README에 provider 비교표, credential이 필요 없는 입력·출력 예제와 PyPI 월간 download badge를 추가합니다.
- download badge는 PSF 공개 데이터 기반 pypistats가 제공하며 알려진 mirror를 제외하고 하루 한 번 갱신됩니다.
